# Релиз Sober Path v1.11.0 🔊

## 🚀 Новые функции

### 1. SOS-аудио и экстренная поддержка 🆘
- Создан новый раздел **«SOS-аудио»** в библиотеке звуков.
- Добавлено **11 управляемых практик** для купирования сильной тяги:
    - Техника заземления 5-4-3-2-1
    - Серфинг тяги (Urge Surfing)
    - Дыхание 4-7-8 и Квадратное дыхание
    - HALT-проверка базовых потребностей
    - Техника СТОП
    - Письмо из будущего
    - И другие.
- Все практики снабжены **голосовым сопровождением** (TTS) для удобства использования в стрессовых ситуациях.

### 2. Тактильный отклик (Haptics) 📳
- Интегрирована система вибрации для улучшения обратной связи:
    - Подтверждение отметки трезвого дня (Success feedback).
    - Предупреждение при отметке срыва (Error feedback).
    - Легкий отклик при отправке сообщений ИИ-коучу.
    - Средний отклик при запуске SOS-практик.

### 3. Интеграция SOS-помощи ⚡
- В компонент экстренной помощи добавлена кнопка быстрого перехода к SOS-аудио.
- Улучшена навигация между разделами при возникновении трудностей.

## 🎨 UI/UX Улучшения
- Обновлен дизайн карточек в разделе звуков.
- Добавлены анимации появления элементов в чате.
- Категория "SOS" теперь выделена ярким красным цветом для быстрой идентификации.

## 🛠 Технические изменения
- Версия приложения обновлена до **1.11.0**.
- Добавлен сервис `sosService.ts` для централизованного управления экстренным контентом.
- Оптимизирована производительность списков в разделе звуков.
