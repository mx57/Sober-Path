# Release Notes - v1.7.0 (Sober Path Rebrand)

## 🎨 Ребрендинг и Визуальное обновление
- Приложение официально переименовано в **Sober Path**.
- Обновлен логотип и заставка (Splash Screen).
- Внедрена новая цветовая схема "Sober Path Green" во всем приложении.
- Добавлены Skeleton-анимации для плавной загрузки контента в Сообществе и ИИ-Чате.

## 🤖 Улучшение ИИ-Коуча
- Исправлен критический баг (crash) при запуске ИИ-Чата.
- Улучшена контекстность ответов: ИИ теперь учитывает ваш стаж трезвости и помнит прошлые достижения.
- Оптимизирован интерфейс чата для лучшей читаемости.

## 💬 Сообщество и Контент
- Реализована возможность создания собственных постов в сообществе.
- Добавлена имитация системы комментариев для поддержки участников.
- Расширена база контента: +3 истории успеха, +6 постов поддержки.
- Внедрена фильтрация ленты по кругам поддержки (Мотивация, Вопросы, Достижения).

## 🛠 Технические изменения
- Настроен Automerge для ускорения процесса разработки.
- Оптимизирована производительность списков.
- Исправлены ошибки типизации и зависимости в хуках.
