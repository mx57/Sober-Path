# Release Notes - v1.13.0 ⚡

## 🤖 Продвинутый ИИ-Ассистент
- **Интерактивные упражнения:** Внедрена система пошаговых упражнений прямо в чате. Теперь Коуч может проводить пользователя через технику заземления 5-4-3-2-1 с визуализацией прогресса.
- **Улучшенный UX диалога:** Добавлены индикаторы выполнения шагов и контекстные подсказки для управления упражнениями.

## 💬 Сообщество и Геймификация
- **Общие цели:** Реализована концепция глобальных целей сообщества (например, «10000 общих дней трезвости»).
- **Виджет прогресса:** В раздел Сообщество добавлен анимированный виджет, показывающий вклад всех участников в достижение общих целей.

## 📚 Контент и Обучение
- **База знаний:** Добавлено 5 новых статей о питании, нейробиологии поддержки и техниках управления стрессом.
- **Новый курс:** Запущен микро-курс «Эмоциональный интеллект», обучающий распознаванию и управлению чувствами.

## 🎨 Интерфейс и UX
- **Интерактивный Недельный План:** Задачи в виджете на главном экране теперь кликабельные — отмечайте выполнение, не переходя в другие разделы.
- **Тактильная отдача:** Улучшена система Haptics для подтверждения выполнения задач и завершения упражнений.
