# Release Notes - v1.15.0

## 🤖 AI-Коуч и Психология
- **Когнитивный рефрейминг:** Добавлено новое интерактивное CBT-упражнение для работы с негативными мыслями прямо в чате.
- **Улучшенная навигация в чате:** Добавлены новые быстрые команды для запуска упражнений.

## 💬 Сообщество и Коммуникация
- **Виджет "Пульс сообщества":** Живая лента активности (сообщения, реакции, достижения) делает приложение более динамичным.
- **Бейджи Наставников:** Опытные участники (стаж 100+ дней) теперь выделяются специальным значком верификации в ленте.
- **Статус наставника:** В посты добавлена информация о стаже трезвости автора.

## 📚 Контент и Обучение
- **Новый микро-курс:** "Осознанное предотвращение срывов" — 3 урока о распознавании предвестников рецидива и создании плана действий.
- **SOS-поддержка:** Добавлено 5 новых техник ("Якорь", "Черный экран", "Лимонный шок" и др.) для экстренного купирования тяги.

## 🎨 Интерфейс и UX
- **Быстрый доступ:** Кнопка "Дневник" добавлена на главный экран для удобного ведения ежедневных записей.
- **Визуальный стиль:** Обновлен дизайн карточек статистики (StatCard) — улучшены тени и добавлена легкая обводка.
- **Оптимизация:** Улучшена плавность анимаций в разделе Сообщество.

## 🛠 Технические изменения
- Расширена модель данных `SupportPost` для поддержки стажа трезвости.
- Обновлен `AICoachService` для поддержки многошаговых NLP-упражнений.
- Версия повышена до 1.15.0.
