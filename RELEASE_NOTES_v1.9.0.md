# Release Notes v1.9.0 - Sober Path ⚡

Этот релиз сфокусирован на расширении образовательного контента, улучшении проактивности ИИ-ассистента и оптимизации пользовательского интерфейса.

## 🚀 Новые возможности

### 📚 Микро-курсы
- Внедрена система пошагового обучения. Доступны курсы:
  - **Первая неделя: Фундамент**: Критически важные знания для новичков.
  - **Работа с триггерами**: Техники распознавания и нейтрализации тяги.
  - **Пересборка жизни**: Построение здоровых привычек и дофаминовых путей.
- Отслеживание прогресса и визуализация завершенных уроков.

### 🤖 Проактивный ИИ-Ассистент
- **Адаптивные напоминания**: Коуч теперь планирует поддержку на основе вашего состояния. Если зафиксирован высокий уровень стресса, через час придет поддерживающее уведомление.
- **Улучшенный диалог**: Ассистент предлагает техники («HALT», «Серфинг по тяге») и рекомендует статьи прямо в чате.

### 💬 Экспертный Q&A в Сообществе
- Добавлен раздел с ответами от профессиональных наркологов и психологов на самые частые вопросы пользователей.

## 🎨 Улучшения интерфейса и UX
- **Оптимизация навигации**: Футер упрощен. Оставлены только ключевые разделы: Главная, Коуч, Курсы, Статьи и Настройки.
- **Инфо-карточки**: Добавлены виджеты «Совет дня» и цитаты на главный экран.
- **Исправления UI**: Исправлена ошибка отображения списка курсов на некоторых устройствах и в веб-интерфейсе.

## 🛠 Технические изменения
- **Обновление окружения**: Переход на Node.js 20 и Java 17 в процессе сборки.
- **Стабильность**: Настроено полноценное тестирование с моками для AsyncStorage и системы уведомлений.
- **Ассеты**: Исправлены ошибки путей к PNG-ассетам, вызывавшие 404 ошибки при экспорте.
